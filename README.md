# Plan.md

## **Prosjekt: Tangen Torv Restaurant – Fullstack-løsning**

### **Kort beskrivelse**

Jeg skal lage nettsider for restauranten **Tangen Torv**, med informasjon om restauranten, meny, åpningstider og kontaktinfo.
Brukere skal kunne **bestille bord**, og ansatte skal ha en egen side for å se og administrere bestillinger.

---

## **Teknologier**

* Database: MariaDB/PostgreSQL (via Docker + Docker Compose)
* Frontend: HTML, CSS, JavaScript
* Backend (modul 3): Node.js + Express
* GitHub til versjonskontroll
* Dokumentasjon i README.md (+ ev. DATABASE.md)

---

## **Modul 1 – Database**

Jeg lager:

* Et enkelt ER-diagram (meny, bestillinger, ansatte)
* SQL-fil som oppretter tabeller og relasjoner
* SQL-fil med eksempelspørringer (insert booking, hente data osv.)
* Docker-compose for å starte databasen

---

## **Modul 2 – Frontend**

Jeg lager statiske sider med hardkodede data:

* Hjem
* Meny
* Åpningstider
* Kontakt
* Bestillingsside (uten databasekobling ennå)
* Ansattside med dummy-data

---

## **Modul 3 – Backend (etter jul)**

Jeg bygger API i Node/Express som:

* Leser meny fra databasen
* Lagrer bestillinger
* Viser bestillinger til ansatte
